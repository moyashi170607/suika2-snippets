suika2をVScodeで書く人向けのスニペット（コード補完）です。

snippets の方は、省略なし版(例：@背景 ファイル=sample.png 秒=1.5 エフェクト=右カーテン)
snippets-short の方は、省略あり版(例：@bg sample.png 1.5 )
です。

使用したい方のファイル内の「.vscode」を
あなたの作品ファイルの一番上の階層（suika.exeとかがある階層）に入れてください。
（既に.vscodeファイルがある場合は、中のファイルをそこに入れてください）

各コマンド、wms、エフェクト名に対応しています（一部非推奨コマンドや削除予定コマンドなどを省く）
また、type名+Temp　で各種GUIテンプレートを展開できます

suika2はktabata氏およびSuika2 Development Teamの方々によって開発されたノベルゲーム開発エンジンです。

このスニペットに関する不具合はこつ子(@KotukoHumibana)にお伝えください。

英語版についても対応予定です。
各種スニペットの説明部分の翻訳案募集中です
